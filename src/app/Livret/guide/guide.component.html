Guides au usager

Introduction

Bienvenue dans notre guide pour les réunions. Ce document vous aidera à comprendre comment planifier efficacement une réunion.

Instructions de Base

1. Planification de réunion

Utilisez le menu "Réunion" dans la barre de menu puis sélectionnez le sous-menu "Planifier une réunion".

2. Rappel de réunion

Pour des réunions efficaces, il est même temps que la planification de la réunion, ainsi qu'un jour et une heure avant la réunion, par l'envoi d'un email automatiquement ou par contact (manual).

FAQ

Comment planifier une réunion ?

Dans l'onglet "Planifier une réunion", lancer les informations précises (préparer des réunions planifiées) et cliquer sur le bouton planifier

Conseils Pratiques

- Testez votre connexion Internet avant de planifier une réunion pour le rappel de réunion par mail

- Utilisez les e-mails exacts de chaque participant en membre de la DSP est accessible pour le rappel de réunion

Introduction

Ce guide est conçu pour aider les utilisateurs à insérer des données dans le système de manière efficace et correcte.

Instructions de Base

1. Accéder au formulaire de saisie de données

Accédez à la section "données" de la DSP, puis sélectionnez "Créer".

2. Remplir le formulaire de saisie

Complétez les champs obligatoires (*) et les champs optionnels si vous avez les informations nécessaires.

FAQ

Que faire si je rencontre une erreur lors de la saisie des données ?

Veuillez vérifier les champs obligatoires et les remplir correctement.

Puis-je modifier les données après les avoir soumises ?

Oui, vous pouvez modifier les données en accédant à la section "Modifier les données" dans le menu principal.

Conseils Pratiques

- Assurez-vous de vérifier les données avant de les soumettre.

- Veuillez vérifier que vous disposez des autorisations nécessaires pour enregistrer des données dans ce système.
